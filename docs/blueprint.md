# **App Name**: eScotty Tracker

## Core Features:

- Charging Session Logging: Manually log charging sessions with date, start/end time, initial and final percentage.
- Data Storage: Store charging session data in a Firestore database.
- Charging History: View a sortable and filterable history of all charging sessions.
- Charging Summary: Summarize charging data. AI tool extracts insights to show total charging time, average charge gained per session.
- Charging Graph: Generate a visual representation of the charging trend in a graph.

## Style Guidelines:

- Primary color: Dark gray (#333333) for a professional look.
- Background color: Light gray (#F0F0F0) for a clean interface.
- Accent color: Dark blue (#3F51B5) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look, suitable for headlines or body text.
- Use minimalist icons for navigation and data representation.
- Employ a clean, card-based layout for session history.
- Use subtle transitions for loading and displaying data.
